## Creating New Runbooks

1. Copy the [runbook template](./template.md)
2. Choose a `sidebar_position` from the appropriate category range
3. Fill in all required sections
4. Include verification steps for each procedure
5. Link to relevant architecture docs and alerts

## Runbook Standards

Every runbook should include:

- **Quick Links** - Dashboard, alerts, Slack, contacts
- **Prerequisites** - Access requirements with verification commands
- **Procedures** - Step-by-step with expected output
- **Verification** - How to confirm success
- **Troubleshooting** - Common issues and fixes
